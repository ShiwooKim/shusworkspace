# Chatterbox Kids: 프로그램 소개(개요)

### 1. 페이지 개요

- 각 언어별 Chatterbox Kids 프로그램의 학습 철학, 커리큘럼 구성, 콘텐츠 유형, 학습 효과를 명확하고 직관적으로 전달하는 **설명 중심 콘텐츠 페이지**
- 상품별로 독립된 페이지로 제공되며, 고객이 학습자의 교육 콘텐츠를 비교/이해하고 선택할 수 있도록 설계

### 2. 페이지 구성

| 페이지명 | 설명 |
| --- | --- |
| **영어 프로그램 소개** | 채터박스키즈의 핵심 프로그램. 3~4년 과정, 말하기 중심, Fun Study & Flow 방식, 대표 콘텐츠 상세 소개 포함 |
| **중국어 프로그램 소개** | 영어 프로그램과 짝을 이루는 구조. 같은 어휘/패턴 기반으로 구성되어 언어 확장이 용이함 |
| **수다쟁이 한글 프로그램 소개** | 음운 인식부터 낱글자, 통글자, 어휘력·표현력 확장까지 가능한 체계적 한글 학습. 영상 중심 |

### 3. 기획 방향성

1. **Seed Learning 로고와 글로벌 프로그램 신뢰감**을 상단에 시각적으로 드러낼 것
2. **15분 반복학습 / 실생활 표현 / 낮은 교사 의존도** 등 핵심 포인트를 **3~4개 카드로 정리**
3. 단계별 프로그램 구성(PreK~Lv3)을 표 형태로 정리하여 빠르게 인식 가능하도록
4. 콘텐츠 요소(Chant, Game, Recording 등)는 UI 스크린샷이나 아이콘 조합으로 전달
5. **가정 연계 학습법**은 실제 예시(“영상 따라 부르기”, “복습 게임” 등) 중심으로 간결히 안내

### 4. 경쟁사 분석

- 톡키즈
    - 톡키즈는 **학습 철학 → 원칙 → 활용법 → 시스템 → 신뢰 메시지** 순으로 흐름을 설계
    - “왜 좋은지 → 어떻게 쓰는지 → 얼마나 편한지”를 시각적으로 이해하도록 구성
    - Chatterbox도 **교육 철학 + 구조화된 학습 방법 + 시스템 장점**으로 접근하는 구성을 고려할 수 있음

| 순서 | 구성 항목 | 내용 요약 | UI 구성 특징 | 채터박스 적용 시 참고 포인트 |
| --- | --- | --- | --- | --- |
| 1 | **교육 철학 & 목표** | CLT(의사소통중심 언어교수법), BBL(뇌 기반 학습)이론 기반. 아이의 유창한 의사소통 능력 향상 목표 | 텍스트 + 배경 이미지 위 키워드 강조 문구 (이론명 강조) | 채터박스도 ‘Fun Study & Flow’ 등의 학습 철학을 상단에 명시 |
| 2 | **프로그램 핵심 포인트 4가지** | 1) 영어노출 극대화2) 느리지만 확실한 반복3) 배경지식 강화4) 주제중심 커리큘럼 | 네 개의 박스 또는 아이콘 그룹으로 시각화 | 채터박스만의 “핵심 학습 원칙 3~5가지” 정리해 시각화 가능 |
| 3 | **활용 방법 안내** | 한 달 2권 학습, 하루 15분 반복, 몸과 소리로 기억하는 학습법 등 | 문장 강조 카드 + 일러스트 (아이/선생님 활용 장면 삽입) | 채터박스의 “짧고 자주 학습” 구조를 학습 루틴으로 시각화 |
| 4 | **스마트 연계** | 멀티 디바이스 학습, 가정 연계, 마이페이지 기능, 뱃지/별 시스템 | 장치 아이콘 + UI 캡처 이미지 (별/배지, 학습이력) | 채터박스도 마이페이지 기능 및 보상 시스템 강조 가능 |
| 5 | **전문가 메시지** | 교육 전문가의 추천사 / 프로그램의 검증성 강조 | 사진 + 인용 형태의 강조 블록 | 채터박스의 Casey Malarcher, Paul Nation 등 저자 소개로 대체 가능 |
- 티키토키
    - **학부모 + 교사 신뢰 확보**가 메인 전략
    - 콘텐츠 그 자체보다는 **활용 방법**과 **일상 속 접점**을 강조
    - **브랜드 파워 강조, 학습 루틴 시각화, 가정 연계**에 특화된 구성

| 순서 | 구성 항목 | 내용 요약 | UI 구성 특징 | 채터박스 적용 시 참고 포인트 |
| --- | --- | --- | --- | --- |
| 1 | **브랜드 신뢰 강조 영역**(비상교육 + 티키토키) | “비상교육”이라는 대형 교육회사 브랜드를 메인 배너에서 강하게 드러냄 | **풀스크린 메인 이미지** + 로고/슬로건 강조 (교실 사진 또는 캐릭터 활용) | 채터박스도 “Seed Learning USA” 등 **해외 개발사 신뢰도 강조** 필요 |
| 2 | **핵심 포인트 4가지**(매일 10분! 영어 습관) | ① 매일 반복 학습 습관 ② 표준보육/누리과정 연계 ③ 담임선생님과 함께하는 일상형 영어 ④ 교육부 초등권장 표현 기반 | 4개의 **텍스트 카드 또는 아이콘 박스** 구성 | 채터박스도 “15분 반복학습 / 유치원 교사 누구나 진행 가능 / 실생활 표현 기반” 등으로 각색 가능 |
| 3 | **커리큘럼 소개**(Toddler ~ Level3) | 연령별 4단계 구성. 각 단계별 교재 수, 학습 내용은 표 또는 구조도로 구성 가능 | 슬라이드 구조 내 제목 강조 + 레벨 카드 | 채터박스도 **단계 명칭 표기 체계 정비 필요** |
| 4 | **교실 수업 방식 소개**(“원에서 이렇게 배워요!”) | 챈트, 애니, 파닉스, 율동 등 콘텐츠 유형 안내이미지 연상 학습법 강조 | 콘텐츠 요소를 이미지+텍스트로 나열, **아이들 사진 활용 중심** | 채터박스 콘텐츠 구성 요소(Chant/Song/Game/Recording 등)도 **시각적으로 정리해서 설명 필요** |
| 5 | **가정 연계 가이드**(“가정에서 이렇게 도와주세요”) | 챈트 따라 부르기, 단어카드 놀이, 확장활동 등 실천법 제시 | 간단한 설명문 + 일러스트 카드 구성 | 채터박스의 **“집에서도 학습하는 영상 + 게임 콘텐츠”**를 활용법 중심으로 안내 가능 |
